Catmull-Rom Spline Generator
COMP 371 Assignment 2
Concordia University
March 9th, 2017
Mathieu Leblanc

Description:
Creates a mesh generated from the translation or rotation of a Catmull-Rom spline. 
The splines are generated from user input control points, and the final mesh is displayed.

Input Controls:

Backspace: 	Restart program
Tab:		Toggle distance or curvature based spline interpolation
Left-Click:	Set a spline control point
Enter:		Generate spline, Proceed

Camera Controls:

Mouse:		Look (Pitch/Yaw)
Mouse-Scroll:	Zoom
W:		Move forwards
A:		Strafe left
S:		Move backwards
D:		Strafe right
Q:		Roll left
E:		Roll right
Space:		Strafe up
Ctrl:		Strafe down
Shift:		[Hold] Sprint
Alt:		[Hold] Walk
